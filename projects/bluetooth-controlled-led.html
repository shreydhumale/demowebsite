Bluetooth Controlled LED with HC-05

Published: April 24, 2025 Bluetooth, Arduino, IoT

Introduction

This project demonstrates how to control an LED using Bluetooth communication. We’ll be using an Arduino board and an HC-05 Bluetooth module to receive commands from a smartphone app and switch the LED on or off. It’s a great starter project for learning wireless control and IoT basics.

The project involves setting up serial communication between the Bluetooth module and Arduino, and using a mobile app to send simple ON/OFF commands. When a command is received, the Arduino turns the LED on or off based on the command.

Components Required

Arduino Uno ×1

Bluetooth Module HC-05 ×1

LED ×1

Smartphone with Bluetooth Controller App

SOURCE CODE 


    // Bluetooth LED Control using HC-05 and Arduino
    char data = 0;
    int ledPin = 13;
    
    void setup() {
      pinMode(ledPin, OUTPUT);
      Serial.begin(9600); // HC-05 default baud rate
    }
    
    void loop() {
      if (Serial.available()) {
        data = Serial.read();
        if (data == '1') {
          digitalWrite(ledPin, HIGH); // LED ON
        } else if (data == '0') {
          digitalWrite(ledPin, LOW); // LED OFF
        }
      }
    }
