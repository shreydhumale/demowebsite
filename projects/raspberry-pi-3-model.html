Raspberry Pi 3 Model B+

Published: April 24, 2025 Raspberry Pi, Linux, IoT

Introduction

The Raspberry Pi 3 Model B+ is a powerful single-board computer that offers an excellent platform for learning, prototyping, and even deploying IoT applications. With its quad-core processor, built-in Wi-Fi, and multiple GPIO pins, it's ideal for a wide range of projects—from media centers to automation systems.

In this project guide, we’ll go through the initial setup of Raspberry Pi 3 Model B+, installing the operating system, and running a simple LED blink program using Python to familiarize ourselves with GPIO handling.


Board Layout

Components Required

Raspberry Pi 3 Model B+ ×1

Micro SD Card (8GB or more) ×1

5V 2.5A Power Supply ×1

LED ×1

220 ohm Resistor ×1

Breadboard and Jumper Wires

SOURCE CODE 


    # Import required libraries
    import RPi.GPIO as GPIO
    import time
    
    # Set GPIO mode
    GPIO.setmode(GPIO.BCM)
    GPIO.setwarnings(False)
    
    # Set GPIO pin 18 as output
    led_pin = 18
    GPIO.setup(led_pin, GPIO.OUT)
    
    # Blink the LED
    print("LED blinking. Press Ctrl+C to stop.")
    try:
        while True:
            GPIO.output(led_pin, GPIO.HIGH)
            time.sleep(1)
            GPIO.output(led_pin, GPIO.LOW)
            time.sleep(1)
    except KeyboardInterrupt:
        print("Program stopped.")
        GPIO.cleanup()
