GSM Module (SIM800L)

Published: April 29, 2025 GSM, SIM800L, IoT, Communication

Introduction

The GSM module (such as SIM800L or SIM900) enables microcontrollers to send and receive SMS, make or receive calls, and even connect to the internet using GPRS. It's widely used in IoT applications like home security systems, weather stations, and GPS tracking.

In this project, we will use the GSM SIM800L module with an Arduino to send a text message when a button is pressed.


Module View

Components Required

GSM Module SIM800L ×1

Arduino UNO ×1

Push Button ×1

Resistors and Jumper Wires

SIM Card (with balance) ×1

Power Supply (3.7V Li-ion or 5V Regulated)

SOURCE CODE


      #include 
      
      SoftwareSerial gsm(7, 8); // RX, TX
      const int buttonPin = 2;
      int buttonState = 0;
      
      void setup() {
        pinMode(buttonPin, INPUT);
        gsm.begin(9600);
        Serial.begin(9600);
        delay(1000);
        gsm.println("AT"); // Check communication
        delay(1000);
      }
      
      void loop() {
        buttonState = digitalRead(buttonPin);
      
        if (buttonState == HIGH) {
          gsm.println("AT+CMGF=1");    // Text mode
          delay(1000);
          gsm.println("AT+CMGS=\"+911234567890\""); // Replace with your number
          delay(1000);
          gsm.print("Button pressed! Sending SMS from GSM module.");
          delay(100);
          gsm.write(26); // ASCII code of CTRL+Z to send message
          delay(5000);
        }
      }
